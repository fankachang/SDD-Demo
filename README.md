# SDD-Demo

本專案為 Spec‑Driven Development（SDD）示範範例，示範使用 `specify` 工具與 AI Agent workflow 來建立需求、檢查清單與發佈流程。

主要目的
- 範例化一套由規格驅動的開發流程（specs → tasks → implement）。
- 展示如何使用 `/specify`, `/clarify`, `/plan`, `/tasks`, `/implement` 等 AI‑agent 工具鏈進行協作。

快速開始
- 已在專案根目錄執行 `specify init` 並自動套用範本（copilot sh template）。
- 建議後續步驟：建立並編輯憲法文件、需求說明，然後使用 agent 指令產出執行計畫與任務。
